What are Data Structures?


Array, one of the most basic data structures

Data structures are specific formats used to efficiently store data that allow for various operations to be performed on said data. Each data structure functions differently from the rest and comes with its own advantages and disadvantages. As a result, some data structures are more suited for specific tasks than others. It is up to the programmer to accurately discern which one is most appropriate for their needs.

Why are they Important?


The ability to choose the most suitable data structure for a project is essential to becoming a quality software developer. Data structures are the building blocks of a program, and choosing the wrong ones will lead to said program running extremely inefficiently. Let's say, for example, you have a collection of integer values and you wish to find the location of a specific one. If you store that data in an array, you would need go through it blindly one value at a time until you find your target. In the worst case scenario, you may have to check every value in the array before you succeed.
