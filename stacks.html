What is a Stack?


A stack is a container in which elements are added and removed in a particular order. For a stack, this order is LIFO, or last in first out. The element that was last added to the stack will be the first to be removed. In order to uphold this rule, stacks only allow for direct insertion and deletion at the top of the stack. When an element is added to the top of a stack, it is called pushing. When one is removed, it is called popping
